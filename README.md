# rest-API
Selected a public REST API that returns structured JSON data such as users, products, or statistics.
Designed a dashboard layout using HTML and CSS with sections for listing, details, and filters.
Used the Fetch API to retrieve data asynchronously from the REST endpoint.
Parsed the JSON response and map the data into meaningful Ul components.
Implemented filtering and sorting features using JavaScript array methods.
Added pagination or lazy loading for better performance simulation.
Handled API errors gracefully and display user-friendly error messages.
